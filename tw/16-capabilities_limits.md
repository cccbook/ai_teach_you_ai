# 16. AI 擅長的事與缺陷

## AI 真正擅長什麼？

經過前面的學習，我們知道 AI（尤其是 LLM）的核心能力：

### 1. 文字生成

LLM 是「下一個 token 預測」機器，天生擅長生成文字。

```python
# AI 擅長：根據風格、話題自動生成流暢的文字
prompt = "以村上春樹的風格寫一段關於雨的文章"
```

### 2. 語言理解

Transformer 的注意力機制讓模型能理解複雜的語意關係：

- 翻譯
- 摘要
- 問答
- 情緒分析

### 3. 程式設計

有趣的是，AI 對程式碼的「理解」有時比對自然語言更精確：

```python
# AI 可以：
# 1. 寫程式碼
# 2. 解釋程式碼
# 3. 找出 bug
# 4. 優化效能
```

### 4. 推理與分析

經過足夠規模的訓練，LLM 展現出複雜推理能力：

- 數學解題
- 邏輯推理
- 因果分析

### 5. 知識整合

模型從海量資料中學習，能快速整合不同領域的知識：

```
問題：「相對論和量子力學有什麼衝突？」
AI 能結合物理、數學、哲學的知識來回答
```

## AI 的缺陷

### 1. 幻覺（Hallucination）

AI 可能產生看似合理但實際錯誤的內容：

```python
prompt = "列舉五個在 1900 年獲得諾貝爾物理學獎的科學家"

# AI 可能回答（部分是錯的）：
# 1. Wilhelm Röntgen（1901 年才是第一屆）
# 2. Hendrik Lorentz
# 3. Pieter Zeeman
# 4. Marie Curie（1903 年，不是 1900 年）
# 5. Henri Becquerel（1903 年）
```

原因：模型是在「預測」最可能的文字，不是在「查詢」事實資料庫。

### 2. 數學計算

LLM 在數學上出人意料地弱：

```python
prompt = "計算 3729 * 4823 等於多少？"

# AI 可能算錯，尤其是大數字
# 因為數學需要精確，不是「最可能的數字」
```

### 3. 實時資訊

模型無法知道訓練截止後發生的事：

```python
# 模型不知道：
# - 最新聞事件
# - 今天的天氣
# - 即時股價
```

### 4. 長期規劃

AI 在單一步驟推理上表現不錯，但長期規劃仍有挑戰：

```
問題：「規劃一個為期一年的減肥計畫，包含每個月的目標和具體執行方法」

問題：目標可能合理，但每個月的銜接、意外情況的處理可能不連貫
```

### 5. 空間推理

理解三維空間關係對 AI 來說很困難：

```python
prompt = "想像一個立方體，如果我從前面移到上面，現在哪個面朝向我？"

# AI 可能回答錯誤
```

## 為什麼會有這些缺陷？

### 1. 訓練目標的限制

下一個 token 預測的目標：
- 鼓勵流暢性 > 準確性
- 鼓勵看似合理 > 真正正確

### 2. 資料的偏差

訓練資料中的錯誤、偏見會被模型學到：

| 偏差類型 | 例子 |
|---------|------|
| 性别偏見 | 「工程師」自動聯想為男性 |
| 文化偏見 | 以西方視角為主的知識 |
| 時代偏見 | 過時的社會規範 |

### 3. 注意力機制的限制

Transformer 的注意力是**固定上下文長度**的：
- 無法處理超出長度的長期依賴
- 會遺忘上下文中較遠的資訊

## 如何與 AI 協作？

了解 AI 的優缺點，才能更好地使用它：

| 適合交給 AI | 需要人類把關 |
|-----------|------------|
| 草稿生成 | 事實查證 |
| 初稿潤飾 | 數學計算 |
| 頭腦風暴 | 即時資訊 |
| 翻譯 | 專業領域判斷 |
| 解釋複雜概念 | 法律/醫療建議 |

## 最佳實踐

```python
# ✅ 善用 AI 的長處
prompt = """
請幫我潤飾以下段落，使其更加流暢優美：
[你的文字]
"""

# ✅ 要求 AI 標記不確定的地方
prompt = """
回答時，對於你不確定的資訊，請用 [不確定] 標記。
問題：維生素 C 是哪一年被發現的？
"""

# ✅ 拆分複雜任務
prompt1 = "請分析這篇文章的主要論點"
prompt2 = "請評估這些論點的邏輯嚴謹性"
prompt3 = "請提出改進建議"
```

## 總結

這一章的核心概念：
- **擅長**：文字生成、語言理解、程式設計、推理、知識整合
- **缺陷**：幻覺、數學計算、實時資訊、長期規劃、空間推理
- **原因**：訓練目標、資料偏差、注意力機制限制
- **協作**：善用 AI 長處，人類把關關鍵領域
