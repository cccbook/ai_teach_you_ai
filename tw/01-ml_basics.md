# 1. 機器學習的第一步——從預測到學習

## 什麼是機器學習？

機器學習是讓電腦從資料中學習規律，然後用學到的規律做預測或決策。與傳統程式設計不同：

| 傳統程式設計 | 機器學習 |
|-------------|---------|
| 規則 → 輸入 → 輸出 | 輸入 → 輸出（資料）→ 規則 |
| 人類寫明確保則 | 機器自動找出規則 |

## 一個簡單的預測問題

想像你要預測房價。給你以下資料：

| 坪數 | 房價（萬） |
|-----|-----------|
| 20  | 400       |
| 35  | 700       |
| 50  | 1000      |
| 80  | 1600      |

你有沒有觀察到規律？

$$房價 ≈ 坪數 × 20$$

這就是機器學習在做的事：從資料中找出規律。

## 監督式學習的基本流程

```
輸入資料 → 特徵提取 → 模型預測 → 與正確答案比較 → 調整模型 → 預測
```

### 1. 輸入與輸出

- **輸入（X）**：你想用來預測的資訊（坪數、年齡、圖片...）
- **輸出（y）**：你要預測的目標（房價、類別、下一個字...）

### 2. 模型

模型就是一個數學函數，把輸入轉換成輸出：

```
y = f(x)
```

簡單的線性模型長這樣：

$$y = w × x + b$$

- `w`：權重（weight），決定輸入的重要程度
- `b`：偏差（bias），調整輸出的基準線

### 3. 損失函數

模型預測後，需要知道「錯得多離譜」：

$$L = (預測值 - 真實值)^2$$

這叫均方誤差（Mean Squared Error）。預測越準確，損失越低。

## 從「不會」到「會」的過程

一開始，模型的預測很爛：

```
真實：400  預測：50   損失：122500
真實：700  預測：120  損失：336400
真實：1000 預測：200  損失：640000
```

經過不斷調整（這叫「訓練」），預測越來越準：

```
真實：400   預測：398   損失：4
真實：700   預測：702   損失：4
真實：1000  預測：998   損失：4
```

## AI 模型是更複雜的函數

線性模型太簡單，處理不了複雜問題。

現代 AI 用的是**神經網路**，本質上也是一個複雜的函數：

$$y = f_{神經網路}(x)$$

只是這個函數有：
- 非常多參數（數百萬到數千億個）
- 多層結構（「深度」學習的由來）
- 非線性運算（能學習複雜模式）

## 總結

這一章的核心概念：
- 機器學習：從資料中學習規律
- 監督式學習：輸入 → 模型 → 輸出，與正確答案比較
- 模型的本質：複雜的數學函數
- 損失函數：衡量預測的錯誤程度
